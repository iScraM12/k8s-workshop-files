# k8s-workshop-files
Contains the necessary files for my k8s workshop starting 2025

## Download a pre-built Windows binary

See [download](https://dl.k8s.io/v1.34.2/bin/windows/amd64/kubectl.exe), download starts, put kubectl.exe somewhere in your PATH. Preferably $HOME

See [releases](https://github.com/helm/helm/releases) download helm.exe, and put it somewhere in your PATH. Preferably $HOME
